SQL Statement Fundamentals

	**SELECT**
	SELECT column_name FROM table_name
	SELECT c1,c2 FROM table_1
	SELECT * FROM table_1
	İstenilen sırada sonuc uretilir.

	**SELECT DISTINCT**
	SELECT DISTINCT column FROM table
	SELECT DISTINCT(column) FROM tabe
	*** unique first element in the column

	**COUNT**
	SELECT COUNT(name) FROM table;
	returns the number of rows in the table.
	SELECT count(choice) FROM table;
	SELECT COUNT(*) FROM table;
	all returns same thing
	SELECT COUNT(DISTINCT name) FROM table;
	SELECT COUNT(DISTINCT(name)) FROM table;

	**SELECT WHERE**
	SELECT column1, column2
	FROM table
	WHERE conditions;
	Immediately after the FROM !
	=
	>
	<
	>=
	<=
	<> veya !=
	AND
	OR
	NOT
	Use single quote  '' for strings.
	* asteriks tüm columnları seçer.
	Rowları saymak icin COLUMN adını COUNT parantezine almalıyız.

	**ORDER BY**
	SELECT col1,col2
	FROM table
	ORDER BY col1 ASC / DESC;
	ORDER BY is at the end of query.
	ASC as default.
	SELECT * FROM customer
	ORDER BY store_id DESC, first name ASC;
	filtrelemelerden sonra yazilir.

	**LIMIT**
	limit the num of rows returned for query.
	show top few rows.
	LIMIT EN SONDA YER ALIR.
	LIMIT 5;
	SELECT * FROM payment
	LIMIT 1;
	veri tiplerini kontrol etmemize yarar.
	limit kullanırken dikkatli ol en kısalarını bulamayabilirsin.

	**BETWEEN**
	value BETWEEN low AND high
	same with:
	value >= low and value <= high
	NOT BETWEEN is also possible.
	and its same with:
	value < low OR value > high
	BETWEEN can also be used with dates.
	'yyyy-mm-dd' format
	normally between is inclusive.
	datetime starts at 0:00 !
	WHERE amount BETWEEN 8 AND 9;
	sondaki güne girer girmez 0:00'da biter !

	**IN**
	WHERE value IN (option1, option2, ... , option_n)
	NOT IN is also possible.

	**LIKE AND ILIKE**
	pattern matching
	use of wildcard characters
	% : matches any seq of chars
	_ : any single char
	WHERE name LIKE 'A%': ALL NAMES 
	THAT BEGIN WITH 'A'
	WHERE name LIKE '%a' : ALL NAMES
	THAT END WITH AN 'a'
	LIKE: CASE-SENSTIVIE
	ILIKE: CASE-INSENSITIVE
	Underscore allows us to replace 
	just a single character
	WHERE title LIKE 'Mission Impossible _'
	Multiple underscore is okay!
	WHERE value LIKE 'Version#_ _'
	We can combine:
	WHERE name LIKE '_her%'
	Devamı: functions-matching.html
	% yerine gelecek ifade bosluk da olabilir.
	Ayrıca LIKE ve ILIKE basina NOT getirilebilir.
